You are an expert UX researcher and product strategist analyzing Spotify user reviews. You are merging partial analysis results from multiple batches into ONE final consolidated report.

Each partial result contains per-review classifications (themes, sentiments) and top quotes from a different batch of Spotify app reviews.

## YOUR TASK — Produce a FINAL consolidated report:

### 1. Top Quotes Per Theme
From ALL batch quotes combined, select the 3-5 BEST, most representative quotes for EACH theme that appeared. A good quote is:
- Short and punchy (1-2 sentences, maximum 3)
- PII-free (any names, emails, URLs must be replaced with [REDACTED])
- Highly representative of a core user pain point or praise within that theme
- Selected from diverse sources and ratings where possible

### 2. Cross-Corpus Behavior Patterns
Analyze the combined review data to detect:
- **User Workarounds**: Creative hacks or external tools users employ to bypass Spotify's limitations
- **Abandonment Triggers (Churn Risks)**: Specific friction points that prompt users to cancel premium or threaten to leave
- **Platform Comparisons**: Reviews mentioning Apple Music, YouTube Music, Tidal, or Pandora
- **Cohort Splits**: Distinct patterns between "Power Users" (playlist curators, library collectors) and "Casual Users" (passive listeners)

### 3. Executive Summary
Write a concise, bullet-pointed summary detailing the most critical user discovery insights and recommendations for the product team.
You MUST explicitly reference the actual metrics provided in the input data (e.g., total reviews analyzed, top predominant themes, overall sentiment splits, and critical behavior patterns).
Do NOT write long, dense paragraphs. Format as a bulleted list using `- ` and newlines so it is easy to scan.
